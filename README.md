# Heart Disease Analysis

## 📌 Project Overview

Heart Disease Analysis is a data analytics project developed using **Tableau** to analyze heart disease data and identify important health trends. The project transforms raw healthcare data into interactive visualizations, dashboards, and stories that help understand the impact of demographic, lifestyle, and medical factors on heart disease.

---

## 🎯 Problem Statement

Heart disease is one of the leading causes of death worldwide. Analyzing large healthcare datasets manually is time-consuming and makes it difficult to identify important risk factors. This project provides an interactive Tableau dashboard to visualize heart disease data and support better healthcare decision-making.

---

## 🎯 Objectives

- Analyze heart disease data using Tableau.
- Identify major risk factors affecting heart disease.
- Create interactive visualizations and dashboards.
- Present insights using Tableau Stories.
- Support healthcare professionals with data-driven decision-making.
